O que o HTML pode fazer?

O HTML é a linguagem de marcação usada para estruturar o conteúdo da web. Com HTML, você pode criar:

  • Páginas da web informativas
  • Formulários interativos
  • Integração de mídia, como imagens e vídeos
  • Hiperlinks para navegar entre páginas
